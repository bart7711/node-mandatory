One of the most inportant thing in formating code is consistency. In Javascript the semicolon at the end of a statement is optional. It is important to choose whether to use it or not and than stick to it.



The general rule in instantiating the variables is using "const" everywhere you can. In cases where the variable needs to be changed "let" should be used.

Some general rules:

- It is better to use dot notation than square bracket notation.

- Instantiating the express server should be done at the top of the file and using app.listen method should be at the bottom
