Express.js is a free and open-source web application framework for Node.js. It is designed for building web applications and APIs and is the standard server framework for Node.js.


How to set up an express server ?


Line 1: We are bringing in the express framework and storing it in a constant.

Line 2: We are initializing the express framework and saving it into another constant.

We can also do it in one line and create only the initialized constant. 

        
            const express = require("express");
            const app = express(); 
            //or
            const app = require("express")();

Making the server run on a specified port

        
            const PORT = 8080;
            app.listen(PORT, () => {
                console.log("Server started on port:", PORT);
            });
