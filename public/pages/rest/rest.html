REST API should have ordered endpoints corenponding with http methods. Like so.


HTTP Method Endpoint What it does
GET /(resource name) Retrieves all the ressources
GET /(resource name)/ID Retrieves a single ressource by ID
POST /(resource name) Create a new ressource
PUT /(resource name)/ID Update a ressource by ID
PATCH /(resource name)/ID Partially updates a ressource ID
DELETE /(resource name)/ID Deletes an existing ressource by ID
